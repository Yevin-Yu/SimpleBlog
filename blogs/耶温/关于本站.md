## 项目简介

**SimpleBlog** 是一个现代化、极简风格的个人博客系统，基于 React 18 + TypeScript + Vite 构建。

### 核心特性

- 🚀 极速加载 - 静态站点生成 (SSG)
- 📱 响应式设计 - 完美适配桌面端和移动端
- 🎨 极简美学 - 简洁优雅的界面
- ⚡ 性能优化 - 代码分割、懒加载
- 🔍 SEO 友好 - 完善的元数据和结构化数据
- 📝 Markdown 支持 - 使用 Markdown 撰写文章
- 🏷️ 标签系统 - 文章分类和标签管理

---

## 技术栈

- **React 18** - 用户界面库
- **TypeScript 5** - 类型安全的 JavaScript
- **Vite 5** - 新一代前端构建工具
- **React Router 6** - 路由管理
- **markdown-it** - Markdown 解析器
- **highlight.js** - 代码高亮
- **DOMPurify** - XSS 防护

## 快速开始

### 环境要求

- Node.js >= 20.11.0
- npm >= 10.0.0

### 安装与运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 写文章指南

### 1. 创建文章

在 `blogs/` 目录下创建 `.md` 文件，支持多级目录：

```
blogs/
├── 耶温/
│   ├── 关于我.md
│   └── 关于本站.md
└── 技术/
    └── React/
        └── Hooks详解.md
```

### 2. Frontmatter 格式

```markdown
---
title: 文章标题
date: 2025-12-28
category: 分类/子分类
description: 文章描述
tags:
  - 标签1
  - 标签2
---
```

**注意**: 文章 ID 会在首次运行时自动生成（8位随机字符）。

### 3. Markdown 语法

支持标准 Markdown 语法：

```markdown
## 标题

正文内容，支持**粗体**和*斜体*。

- 列表项
- 列表项

\```javascript
// 代码块
const hello = 'world';
\```

[链接](https://example.com)
```

## 常见问题

### Q: 如何修改网站标题和描述？

编辑 `src/config/index.ts` 中的 `SITE_CONFIG` 和 `SEO_CONFIG`。

### Q: 如何修改默认文章？

编辑 `src/config/index.ts` 中的 `BLOG_CONFIG.defaultBlogId`。

### Q: 如何修改侧边栏默认展开的分类？

编辑 `src/utils/blog.utils.ts` 中的 `expandedPaths` 配置。

### Q: 文章 ID 是如何生成的？

首次运行时自动生成 8 位随机 ID（小写字母+数字），写入 frontmatter。
